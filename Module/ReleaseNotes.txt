	Date: 8/23/2007
	Desription: 



v02.00.01 - Initial release. 
	Date: 9/27/2006
	Desription: Includes the ability to add and edit products. 
	There is a basic listing control for Products, using a rudementary templating system for display. 
	Currently DnnCart supports the ability to add/edit/delete products. There is no sense of a shopping cart yet.
	A Product currently has the following attributes
		Name
		Short Description
		Long Description
		Quantity
		Cost
		Price
